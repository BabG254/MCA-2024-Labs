Task Overview

This week, the MEI file from Week 3 was modified to include additional elements such as:

  • Titles
  • Composer metadata
  • Additional notes
  • Dynamics (e.g., forte, piano)
  • Annotations

MEI File Structure

The updated MEI file structure with modifications is shown below:

<mei xmlns="https://www.verovio.org/mei-viewer.xhtml?musicxml=true&local=true">
  <meiHead>
    <fileDesc>
      <titleStmt>
        <title>Twinkle Twinkle Little Star - Modified</title>
        <respStmt>
          <resp>Composer</resp>
          <name>Wolfgang Amadeus Mozart</name>
        </respStmt>
      </titleStmt>
    </fileDesc>
  </meiHead>
  <music>
    <body>
      <note pname="c" oct="4" dur="4"/> 
      <note pname="d" oct="4" dur="4"/> 
      <!-- Additional notes and dynamics added -->
    </body>
  </music>
</mei>
