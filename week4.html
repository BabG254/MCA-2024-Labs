Task Overview

This week focuses on analyzing the MIDI file created from the MusicXML score of Week 2. The following tasks were performed:

  1. Exported the MusicXML file from Week 2 as a MIDI file.
  2. Analyzed the MIDI file using JSymbolic.
  3. Generated visualizations (Piano Roll and Pitch Histogram) using the music21 Python library.
