Task Overview

This week, we focused on advanced MEI notations, including:

  • Adding slurs and beams.
  • Incorporating articulations like staccato and legato.
  • Adding dynamic markings for expression.

MEI File Structure

The updated MEI file structure with notations is shown below:

<mei xmlns="https://www.verovio.org/mei-viewer.xhtml?musicxml=true&local=true">
  <meiHead>
    <fileDesc>
      <titleStmt>
        <title>Twinkle Twinkle Little Star - Advanced Notations</title>
      </titleStmt>
    </fileDesc>
  </meiHead>
  <music>
    <body>
      <note pname="c" oct="4" dur="4"/> 
      <note pname="d" oct="4" dur="4" articul="staccato"/> 
      <beam>
        <note pname="e" oct="4" dur="8"/> 
        <note pname="f" oct="4" dur="8"/> 
      </beam>
      <!-- Added dynamics and slurs -->
      <dynamic>p</dynamic>
      <slur><note pname="g" oct="4" dur="2"/></slur>
    </body>
  </music>
</mei>
